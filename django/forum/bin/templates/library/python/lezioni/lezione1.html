Introduzione

Creiamo il nostro primo programma in Python.
Apriamo un editor di testo e scriviamo il seguente codice:
print("Hello World!")
Salviamo il file con estensione .py e lo eseguiamo da terminale
Su OS Linux:
$ python3 nomefile.py
oppure su OS Windows:
$ python nomefile.py
Se tutto è andato a buon fine, dovremmo vedere la scritta Hello World! stampata a schermo.
Congratulazioni! Hai appena scritto il tuo primo programma in Python!

Python Shell

Python Shell è un ambiente interattivo che permette di scrivere codice Python e vederne l'output immediatamente.
Per avviare Python Shell, apri il terminale e scrivi
Su OS Linux:
$ python3
oppure su OS Windows:
$ python
Una volta avviato Python Shell, puoi scrivere il codice Python direttamente nel terminale e premere Invio per eseguirlo.

Python Syntax

In Python, l'indentazione è molto importante.
Mentre in altri linguaggi di programmazione l'indentazione è solo una questione di stile, in Python è obbligatoria.
Ad esempio, il seguente codice genererà un errore:
if 5 > 2:
print("Ciao a tutti!")
Per risolvere il problema, basta indentare la seconda riga:
if 5 > 2:
    print("Ciao a tutti!")
Ora il codice funzionerà correttamente.

Commenti

I commenti sono utili per rendere il codice più leggibile.
In Python, i commenti iniziano con il simbolo # e terminano alla fine della riga.
Ad esempio:
# Questo è un commento
print("Hello World!")
