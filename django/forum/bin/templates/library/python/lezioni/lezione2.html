Input e output

In Python, le operazioni di input e output sono gestite dalle funzioni input() e print().
La funzione input() permette di acquisire dati da tastiera e restituisce una stringa.
Esempio:
nome = input("Inserisci il tuo nome: ")
Quando eseguiamo il codice, il programma si ferma e attende che l'utente inserisca, in questo caso, il proprio nome.
La variabile nome conterrà il valore inserito dall'utente.
La funzione print() permette di stampare a schermo il valore di una variabile o una stringa.
Esempio:
print("Ciao, " + nome)
Questo codice stamperà a schermo il messaggio "Ciao, " seguito dal nome inserito dall'utente.
Esempio completo:
nome = input("Inserisci il tuo nome: ")
print("Ciao, " + nome)

Decisioni e flusso di controllo

Le decisioni in Python sono gestite dalle istruzioni if, elif e else, Python non ha l'istruzione switch.
Esempio:
if condizione:
    # blocco di codice
elif condizione:
    # blocco di codice
else:
    # blocco di codice
Se la condizione dell'istruzione if è vera, viene eseguito il blocco di codice corrispondente.
Se la condizione dell'istruzione elif è vera, viene eseguito il blocco di codice corrispondente.
Se nessuna delle condizioni precedenti è vera, viene eseguito il blocco di codice dell'istruzione else.
Esempio completo:
x = 10
if x > 0:
    print("x è positivo")
elif x < 0:
    print("x è negativo")
else:
    print("x è zero")

Cicli

I cicli in Python sono gestiti dalle istruzioni for e while.
Il ciclo for viene utilizzato per iterare su una sequenza di elementi.
Esempio ciclo for con range:
for i in range(5):
    print(i)
Il ciclo for itera su una sequenza di numeri da 0 a 4 e stampa ciascun numero.
Esempio ciclo for con list:
frutta = ["mela", "banana", "pera"]
for f in frutta:
    print(f)
In questo caso, il ciclo for itera su una lista di frutta e stampa ciascun elemento.
Sia con range che con list, il ciclo for itera su tutti gli elementi della sequenza,
con la differenza che con range si itera sugli indici della sequenza, mentre con list si itera sugli elementi stessi.
Il ciclo while viene utilizzato per eseguire un blocco di codice finché una condizione è vera.
Esempio:
i = 0
while i < 5:
    print(i)
    i += 1
Il cliclo itera finché la variabile i è minore di 5,
ad ogni iterazione viene stampato il valore di i e incrementato di 1.
Esistono delle istruzioni di controllo che permettono di modificare il flusso di un ciclo:
  • break: interrompe il ciclo e esce dal blocco di codice
  • continue: salta il resto del blocco di codice e passa alla successiva iterazione
Esempi:
for i in range(10):
    if i == 5:
        break
    print(i)
Questo ciclo for stampa i numeri da 0 a 4 e si interrompe quando i è uguale a 5.
for i in range(10):
    if i == 5:
        continue
    print(i)
Questo ciclo for stampa tutti i numeri da 0 a 9, tranne il numero 5.

Funzioni

Le funzioni sono blocchi di codice che possono essere riutilizzati in diversi punti del programma.
Puoi passare parametri a una funzione e restituire un valore o più valori.
In Python, le funzioni, sono definite con la parola chiave def.
def saluta():
    print("Ciao")
Per chiamare una funzione, basta scrivere il suo nome seguito da parentesi tonde.
def saluta():
    print("Ciao")

saluta()
Per passare parametri a una funzione, basta inserirli tra le parentesi tonde della definizione.
def saluta(nome):
    print("Ciao, " + nome)

saluta("Mario")
Puoi passare più parametri separandoli con una virgola.
def saluta(nome, cognome):
    print("Ciao, " + nome + " " + cognome)
Oppure puoi passare i parametri con il loro nome.
saluta(cognome="Rossi", nome="Mario")
Così facendo, l'ordine dei parametri non è importante.
E' anche possibile stabilire un valore di default per i parametri di una funzione.
def saluta(nome="Anonimo"):
    print("Ciao, " + nome)

saluta()
Così facendo, se non viene passato alcun parametro alla funzione, verrà utilizzato il valore di default.
Le funzioni possono restituire un valore con la parola chiave return.
def somma(a, b):
    return a + b

risultato = somma(2, 3)
La funzione somma restituirà la somma dei due parametri passati.
E' possibile restituire più valori separandoli con una virgola.
def operazioni(a, b):
    somma = a + b
    differenza = a - b
    return somma, differenza

s, d = operazioni(5, 3)

Funzioni lambda

Le funzioni lambda sono funzioni anonime definite con la parola chiave lambda.
Sono utili per definire funzioni semplici in una sola riga.
Esempio:
doppio = lambda x: x * 2 print(doppio(5))
Questo codice definisce una funzione lambda che raddoppia il valore passato come parametro.

Funzioni ricorsive

Le funzioni ricorsive sono funzioni che si richiamano da sole.
E' importante definire un caso base per evitare che la funzione vada in loop infinito.
Esempio:
def fattoriale(n):
    if n == 0:
        return 1
    else:
        return n * fattoriale(n - 1)

print(fattoriale(5))
Questo codice calcola il fattoriale di un numero utilizzando una funzione ricorsiva.
