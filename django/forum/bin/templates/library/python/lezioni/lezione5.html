Moduli e pacchetti

Un modulo è un file contenente codice Python.
Un pacchetto è una raccolta di moduli.

Creare un modulo

Per creare un modulo, è sufficiente salvare il codice Python in un file con estensione .py.
# modulo.py

def saluto(nome):
print("Ciao, " + nome)

Utilizzare un modulo

Per utilizzare un modulo, è necessario importarlo.
import modulo

modulo.saluto("Anna")
A questo punto, il modulo mymodule verrà importato e la funzione greeting verrà eseguita.
Ciao, Anna

Gestione degli errori

  • try: blocco di codice in cui possono verificarsi errori
  • except: blocco di codice che viene eseguito in caso di errore
  • finally: blocco di codice che viene eseguito alla fine, indipendentemente dal risultato
Python fornisce un'ampia gamma di istruzioni per la gestione degli errori.
try:
print(x)
except NameError:
print("Il valore x non e' definito")
In questo caso, se la variabile x non è definita, verrà generata un'eccezione NameError. Per vedere tutti i tipi di eccezioni, clicca qui.

File e I/O

Python offre diverse funzionalità per la lettura e la scrittura di file.
La chiave per lavorare con i file in Python è la funzione open().
Ci sono diversi modi per aprire un file:
  • "r" - Lettura - Default. Apre un file per la lettura, errore se il file non esiste
  • "a" - Append - Apre un file per l'aggiunta, crea il file se non esiste
  • "w" - Write - Apre un file per la scrittura, crea il file se non esiste, se il file esiste gia', verra' sovrascritto
  • "x" - Create - Crea il file specificato, restituisce un errore se il file esiste
Per leggere un file:
f = open("file.txt", "r")
print(f.read())
Questo codice aprirà il file file.txt in modalità di lettura e stamperà il suo contenuto. Invece, per scrivere su un file:
f = open("file.txt", "w")
f.write("Hello, World!")
In aggiunta, un file puo' essere aperto in due modalita' differenti:
  • "t" - Testo - Default. Modalita' di apertura in modalita' testuale
  • "b" - Binario - Modalita' di apertura in modalita' binaria
Quando si lavora con file, è importante chiudere il file dopo averlo utilizzato.
f = open("file.txt", "r")
print(f.read())
f.close()
Cosi' facendo, si evita di consumare risorse del sistema, e se qualcun'altro vuole accedere al file, non ci saranno problemi.

Espressioni regolari

Le espressioni regolari sono sequenze di caratteri che definiscono un modello di ricerca.
Python offre il modulo 're' per lavorare con espressioni regolari.
import re
txt = "Il sole a Berlino"
x = re.search("^Il.*Berlino$", txt)
Questo codice cercherà una corrispondenza all'inizio della stringa con "Il" e alla fine con "Berlino". Per vedere tutti i metodi disponibili, e i simboli utilizzabili con le re: clicca qui.

String formatting

Python offre diversi metodi per formattare le stringhe.
price = 49
txt = "Il prezzo e' {} dollari"
print(txt.format(price))
Questo codice stamperà "Il prezzo è 49 dollari".
Oppure, si puo' utilizzare il metodo f-string:
price = 49
txt = f"Il prezzo e' {price} dollari"
print(txt)

*args, **kwargs

1. '*args' (Argomenti non chiave): Consente di passare un numero variabile di argomenti a una funzione.
def giovane(*kids):
print("Il piu' giovane e' " + kids[2])

giovane("Emilia", "Tobia", "Luca")
In questo esempio, la funzione 'giovane' accetta un numero qualsiasi di argomenti.
All'interno della funzione, 'args' e' una tupla contenente gli argomenti passati alla funzione. 2. '**kwargs' (Argomenti chiave): Consente di passare un numero variabile di argomenti chiave a una funzione.
def mio_amico(**amici):
print("Il mio migliore amico e' " + amici["best"])

mio_amico(best="Mirco", second="Matteo")
In questo esempio, la funzione 'mio_amico' accetta un numero qualsiasi di argomenti chiave.
All'interno della funzione, 'kwargs' e' un dizionario contenente gli argomenti chiave passati alla funzione. Queste caratteristiche sono molto utili quando non si è sicuri in anticipo di quanti argomenti verranno passati alla funzione, o quando si desidera consentire agli utenti di passare argomenti con nome alla funzione.
