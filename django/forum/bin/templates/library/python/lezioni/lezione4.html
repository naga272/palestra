Classi e oggetti

Python è un linguaggio di programmazione orientato agli oggetti. Una classe è un modello/progetto per creare oggetti.

Creare una classe

Per creare una classe, si utilizza la parola chiave class seguita dal nome della classe.
class Myclass:
x = 5

Creare un oggetto

Una volta creata una classe, è possibile creare un oggetto.
obj1 = Myclass()
print(obj1.x)

Metodi

I metodi sono funzioni definite all'interno di una classe.
Ogni classe ha un metodo __init__ che viene eseguito automaticamente quando l'oggetto viene creato.
Il metodo __init__ è comunemente usato per inizializzare gli attributi dell'oggetto, o altre operazioni necessarie per creare l'oggetto.
class Myclass:
def __init__(self, name, age):
self.name = name
self.age = age

obj1 = Myclass("John", 36)
print(obj1.name)
print(obj1.age)
Il metodo __init__ è chiamato il costruttore della classe.
La parola chiave self è un riferimento all'istanza corrente della classe e viene utilizzata per accedere alle variabili dell'istanza, e si riferisce all'oggetto stesso.
Ci sono anche altri metodi speciali, come __str__, __len__, __del__, __repr__, etc.
Ad esempio, il metodo __str__ restituisce una rappresentazione della classe come stringa.
Senza il metodo __str__, stampando l'oggetto si otterrebbe un output non molto utile, precisamente la posizione in memoria dell'oggetto.
class Myclass:
def __str__(self):
return "Hello, my name is " + self.name

obj1 = Myclass()
obj1.name = "John"
print(obj1)
Ciò può essere utile per stampare informazioni sull'oggetto, senza dover stampare ogni attributo singolarmente. Ogni oggetto può avere metodi.
I metodi sono funzioni definite all'interno della classe.
Essi possono essere chiamati su un oggetto e possono anche avere parametri.
class Myclass:
def __init__(self, name, age):
self.name = name
self.age = age

def myfunc(self):
print("Hello, my name is " + self.name)

obj1 = Myclass("John", 36)
obj1.myfunc()
La funzione myfunc stampa un messaggio con il nome dell'oggetto, se proviamo a chiamare la funzione senza creare l'oggetto, otterremo un errore.

Ereditarietà

La ereditarietà è un concetto importante in programmazione orientata agli oggetti.
Le classi possono ereditare attributi e metodi da altre classi.
Per creare una classe che eredita da un'altra classe, si definisce la classe figlia con la classe padre tra parentesi.
class Person:
def __init__(self, name, age):
self.name = name
self.age = age

def myfunc(self):
print("Hello, my name is " + self.name)

class Student(Person):
pass

obj1 = Student("John", 36)
obj1.myfunc()
La classe Student eredita dalla classe Person.
Questo significa che la classe Student avrà gli stessi attributi e metodi della classe Person.
La funzione pass indica che la classe Student non ha alcun attributo o metodo proprio.

Polimorfismo

Il polimorfismo è un concetto che permette di utilizzare un'interfaccia comune per oggetti di diverse classi.
In Python, il polimorfismo è ottenuto tramite metodi con lo stesso nome in classi diverse.
class Dog:
def speak(self):
return "Woof!"


class Cat:
def speak(self):
return "Meow!"

def get_sound(animal):
return animal.speak()


dog = Dog()
cat = Cat()
print(get_sound(dog))
print(get_sound(cat))
In questo esempio, le classi Dog e Cat hanno un metodo speak.
Il metodo get_sound prende un oggetto come argomento e chiama il metodo speak su di esso.
Questo permette di chiamare il metodo speak su oggetti di classi diverse, ottenendo risultati diversi.

Incapsulamento

L'incapsulamento è un concetto che permette di nascondere i dettagli di implementazione di un oggetto.
In Python, l'incapsulamento è ottenuto tramite l'uso di attributi privati.
Gli attributi privati sono definiti con due trattini bassi davanti al nome dell'attributo.
class Person:
def __init__(self, name, age):
self.__name = name
self.__age = age

def get_name(self):
return self.__name

def get_age(self):
return self.__age


obj1 = Person("John", 36)
print(obj1.get_name())
print(obj1.get_age())
In questo esempio, gli attributi __name e __age sono privati e non possono essere acceduti direttamente dall'esterno della classe.
Per accedere agli attributi privati, vengono definiti metodi get_name e get_age che restituiscono i valori degli attributi.
Questo permette di nascondere i dettagli di implementazione della classe e di proteggere gli attributi da modifiche accidentali.

Conclusione

In questo capitolo abbiamo visto come creare classi e oggetti in Python, come definire metodi all'interno di una classe, come utilizzare l'ereditarietà per creare classi figlie che ereditano attributi e metodi da una classe padre, come utilizzare il polimorfismo per utilizzare un'interfaccia comune per oggetti di diverse classi, e come utilizzare l'incapsulamento per nascondere i dettagli di implementazione di un oggetto.
