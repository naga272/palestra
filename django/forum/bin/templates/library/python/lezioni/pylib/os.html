Classi

Nome Descrizione
class os.DirEntry Rappresenta un'entrata in una directory, fornendo informazioni su file e cartelle in modo efficiente.
class os.PathLike Interfaccia per oggetti che possono essere convertiti in un percorso di file, consentendo una maggiore flessibilità nel passaggio di percorsi.
class os.sched_param(sched_priority) Classe per rappresentare i parametri di pianificazione, inclusa la priorità di un processo.
class os.stat_result Classe per rappresentare il risultato di una chiamata a os.stat(), contenente informazioni sui file come dimensione, data di modifica e permessi.
class os.terminal_size Classe per rappresentare le dimensioni del terminale (larghezza e altezza) in colonne e righe.
