random



Fornisce algoritmi per generare numeri casuali


Funzioni

Nome Descrizione
random.seed(a: Optional[Hashable] = None, version: int = 2) -> None Inizializza il generatore di numeri casuali con un valore di seed. Se non viene fornito, usa un valore di sistema casuale.
random.getstate() -> object Restituisce l'oggetto interno che rappresenta lo stato del generatore di numeri casuali.
random.setstate(state: object) -> None Imposta lo stato del generatore di numeri casuali con un oggetto stato precedentemente salvato.
random.randbytes(n: int) -> bytes Restituisce una stringa di byte casuali di lunghezza n.
random.randrange(start: int, stop: Optional[int] = None, step: int = 1) -> int Restituisce un numero intero casuale tra `start` e `stop-1` con un passo opzionale.
random.randint(a: int, b: int) -> int Restituisce un numero intero casuale tra `a` e `b`, inclusi entrambi gli estremi.
random.choice(seq: Sequence) -> Any Restituisce un elemento casuale da una sequenza non vuota.
random.choices(population: Sequence, weights: Optional[Sequence] = None, *, cum_weights: Optional[Sequence] = None, k: int = 1) -> list Restituisce una lista di scelte casuali con sostituzione da una popolazione. Le probabilità possono essere influenzate da pesi o pesi cumulativi.
random.shuffle(x: list, random: Optional[Callable] = None) -> None Mischia casualmente la sequenza fornita in-place.
random.sample(population: Sequence, k: int, *, counts: Optional[Sequence] = None) -> list Restituisce un nuovo elenco contenente `k` elementi unici selezionati dalla popolazione senza sostituzione.
random.random() -> float Restituisce un numero a virgola mobile casuale tra 0.0 e 1.0.
random.uniform(a: float, b: float) -> float Restituisce un numero a virgola mobile casuale tra `a` e `b`.
random.triangular(low: float = 0.0, high: float = 1.0, mode: float = None) -> float Restituisce un numero casuale secondo una distribuzione triangolare.
random.betavariate(alpha: float, beta: float) -> float Restituisce un numero casuale da una distribuzione Beta con i parametri `alpha` e `beta`.
random.expovariate(lambd: float) -> float Restituisce un numero casuale secondo una distribuzione esponenziale.
random.gammavariate(alpha: float, beta: float) -> float Restituisce un numero casuale secondo una distribuzione Gamma.
random.gauss(mu: float, sigma: float) -> float Restituisce un numero casuale secondo una distribuzione normale (Gaussiana) con media `mu` e deviazione standard `sigma`.
random.lognormvariate(mu: float, sigma: float) -> float Restituisce un numero casuale secondo una distribuzione log-normale.
random.normalvariate(mu: float, sigma: float) -> float Restituisce un numero casuale secondo una distribuzione normale (Gaussiana) con media `mu` e deviazione standard `sigma`.
random.vonmisesvariate(mu: float, kappa: float) -> float Restituisce un numero casuale secondo una distribuzione di Von Mises.
random.paretovariate(alpha: float) -> float Restituisce un numero casuale secondo una distribuzione di Pareto.
random.weibullvariate(alpha: float, beta: float) -> float Restituisce un numero casuale secondo una distribuzione di Weibull.

Costanti

Nome Descrizione
random.TWOPI Una costante che rappresenta il valore di 2 * π (approssimativamente 6.283185307179586).
random.LOG4 Una costante che rappresenta il valore di log(4) (approssimativamente 1.3862943611198906).
random.SG_MAGICCONST Una costante utilizzata internamente per l'algoritmo di distribuzione Gaussiana (σ = 1.0).
random.BPF Una costante che rappresenta il numero di bit per la precisione floating-point (approssimativamente 53).
random.RECIP_BPF Una costante che rappresenta l'inverso di BPF, usata per generare numeri casuali a virgola mobile (approssimativamente 2**-53).

Classi

Nome Descrizione
random.Random La classe principale utilizzata per generare numeri casuali. Fornisce un generatore di numeri casuali pseudo-casuale basato su Mersenne Twister.
random.SystemRandom Una sottoclasse di `Random` che utilizza le fonti di casualità del sistema operativo per generare numeri casuali (ad esempio, `/dev/urandom` su Unix).
