re



Fornisce funzioni per la gestione delle regex.
Le regex (dette anche espressioni regolari o regular esxpression) vengono usate per formare un modello di ricerca, quindi possono essere usate per verificare se all'interno di una stringa c'Ã¨ il modello di ricerca rishiesto.


Funzioni

Nome Descrizione
re.compile(pattern: str | bytes, flags: int = 0) -> Pattern Compila un pattern di espressione regolare in un oggetto Pattern
re.search(pattern: str | bytes | Pattern, string: str | bytes, flags: int = 0) -> Match | None Cerca la prima corrispondenza del pattern nella stringa e restituisce un oggetto Match o None se non trova corrispondenze.
re.match(pattern: str | bytes | Pattern, string: str | bytes, flags: int = 0) -> Match | None Verifica se il pattern corrisponde all'inizio della stringa. Restituisce un Match o None.
re.fullmatch(pattern: str | bytes | Pattern, string: str | bytes, flags: int = 0) -> Match | None Controlla se il pattern corrisponde all'intera stringa. Restituisce un oggetto Match o None.
re.split(pattern: str | bytes | Pattern, string: str | bytes, maxsplit: int = 0, flags: int = 0) -> list[str | bytes] Divide la stringa utilizzando il pattern come delimitatore. Restituisce una lista.
re.findall(pattern: str | bytes | Pattern, string: str | bytes, flags: int = 0) -> list[str | tuple] Restituisce tutte le corrispondenze del pattern nella stringa sotto forma di lista. Se ci sono gruppi di cattura, restituisce una lista di tuple.
re.finditer(pattern: str | bytes | Pattern, string: str | bytes, flags: int = 0) -> Iterator[Match] Restituisce un iteratore sugli oggetti Match per tutte le corrispondenze trovate.
re.sub(pattern: str | bytes | Pattern, repl: str | bytes | Callable[[Match], str | bytes], string: str | bytes, count: int = 0, flags: int = 0) -> str | bytes Sostituisce le corrispondenze del pattern con repl nella stringa. Restituisce la stringa o i byte modificati.
re.subn(pattern: str | bytes | Pattern, repl: str | bytes | Callable[[Match], str | bytes], string: str | bytes, count: int = 0, flags: int = 0) -> tuple[str | bytes, int] Simile a sub(), ma restituisce una tupla con la stringa modificata e il numero di sostituzioni effettuate.
escape(string: str | bytes) -> str | bytes Escapa i caratteri che hanno significato speciale nelle espressioni regolari. Restituisce una stringa o byte.
purge() -> None Svuota la cache interna del modulo. Non accetta parametri e non restituisce nulla.
