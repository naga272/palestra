Liste

Le liste sono una delle strutture dati più utilizzate in Python. Sono sequenze ordinate di elementi, che possono essere di qualsiasi tipo.
Per creare una lista, si utilizzano le parentesi quadre [].
Esempio:
lista = [1, 2, 3, 4, 5]
In questo caso, la variabile lista conterrà una lista di numeri interi.
Come scritto prima, le liste possono contenere elementi di qualsiasi tipo, anche altre liste.
Esempio:
lista = [1, "due", [3, 4], 5]
Qui la lista contiene un intero, una stringa, un'altra lista e un altro intero.
Per accedere agli elementi di una lista, si utilizza l'indice dell'elemento desiderato, gli indici partono da 0.
Esempio:
print(lista[0])
L'output sarà 1, poiché l'elemento con indice 0 è il primo elemento della lista.
Per modificare un elemento di una lista, si utilizza l'indice dell'elemento da modificare.
Esempio:
lista[0] = 10
In questo caso, l'elemento con indice 0 (che era 1) verrà modificato in 10.

Operazioni sulle liste più utilizzate

Operazione Descrizione
append() Aggiunge un elemento in coda alla lista
insert() Aggiunge un elemento in una posizione specifica
remove() Rimuove un elemento dalla lista
pop() Rimuove e restituisce l'elemento in una posizione specifica
clear() Rimuove tutti gli elementi dalla lista
sort() Ordina gli elementi della lista
reverse() Inverte l'ordine degli elementi della lista

Tuple

Le tuple sono simili alle liste, ma sono immutabili, cioè non possono essere modificate dopo la creazione.
Per creare una tupla, si utilizzano le parentesi tonde ().
Esempio:
tupla = (1, 2, 3, 4, 5)
In questo caso, la variabile tupla conterrà una tupla di numeri interi.
Per accedere agli elementi di una tupla, si utilizza l'indice dell'elemento desiderato, gli indici partono da 0.
Esempio:
print(tupla[0])
L'output sarà 1, poiché l'elemento con indice 0 è il primo elemento della tupla.
Anche le tuple possono contenere elementi di qualsiasi tipo, anche altre tuple.
Esempio:
tupla = (1, "due", (3, 4), 5)
Qui la tupla contiene un intero, una stringa, un'altra tupla e un altro intero.
Le tuple sono immutabili, quindi non è possibile modificare i loro elementi.

Operazioni sulle tuple più utilizzate

Operazione Descrizione
count() Conta quante volte un elemento è presente nella tupla
index() Restituisce l'indice di un elemento specifico

Dizionari

I dizionari sono collezioni non ordinate di coppie chiave-valore.
Per creare un dizionario, si utilizzano le parentesi graffe {}.
Esempio:
dizionario = {"chiave1": "valore1", "chiave2": "valore2"}
In questo caso, la variabile dizionario conterrà un dizionario con due coppie chiave-valore.
Per accedere ai valori di un dizionario, si utilizza la chiave corrispondente.
Esempio:
print(dizionario["chiave1"])
L'output sarà valore1, poiché la chiave chiave1 corrisponde al valore valore1.
I dizionari possono contenere valori di qualsiasi tipo, anche altri dizionari.
Esempio:
dizionario = {"chiave1": "valore1", "chiave2": {"chiave3": "valore3"}}
Qui il dizionario contiene una stringa e un altro dizionario, basti pensare ad un file json.
Per modificare un valore di un dizionario, si utilizza la chiave corrispondente.
Esempio:
dizionario["chiave1"] = "nuovo valore"

Operazioni sui dizionari più utilizzate

Operazione Descrizione
get() Restituisce il valore corrispondente a una chiave specifica
items() Restituisce una lista di tuple contenenti tutte le coppie chiave-valore
keys() Restituisce una lista con tutte le chiavi del dizionario
values() Restituisce una lista con tutti i valori del dizionario
pop() Rimuove e restituisce il valore corrispondente a una chiave specifica
popitem() Rimuove e restituisce l'ultima coppia chiave-valore inserita
clear() Rimuove tutte le coppie chiave-valore dal dizionario

Set

I set sono collezioni non ordinate di elementi unici.
Per creare un set, si utilizzano le parentesi graffe {}.
Attenzione, qui stiamo usando le parentesi graffer per un set, non un dizionario, quindi non ci saranno coppie chiave-valore.
Esempio:
insieme = {1, 2, 3, 4, 5}
In questo caso, la variabile insieme conterrà un set di numeri interi.
I set non possono contenere elementi duplicati, quindi se si cerca di aggiungere un elemento già presente, non verrà aggiunto.
Esempio:
insieme = {1, 2, 3, 4, 5, 1}

Operazioni sui set più utilizzate

Operazione Descrizione
add() Aggiunge un elemento al set
remove() Rimuove un elemento dal set
discard() Rimuove un elemento dal set, se presente
pop() Rimuove e restituisce un elemento casuale dal set
clear() Rimuove tutti gli elementi dal set
union() Restituisce un set contenente l'unione di due set
intersection() Restituisce un set contenente l'intersezione di due set
difference() Restituisce un set contenente la differenza tra due set
