Libreria standard di Python

Immagina la libreria standard come un gigantesco toolbox pieno di attrezzi specifici per svolgere diversi lavori.
Ogni attrezzo è un modulo, un file che contiene una serie di funzioni, classi e variabili predefinite che puoi utilizzare direttamente nei tuoi programmi.
Alcuni esempi includono:
Libreria Descrizione
os Interagisce con il sistema operativo, permettendo di creare directory, leggere file, eseguire comandi, ecc.
datetime Manipolazione di date e orari
random Genera numeri casuali.
re Supporta espressioni regolari per la manipolazione di testo.
math Fornisce funzioni matematiche di base come seno, coseno, logaritmi, ecc.
json Converte dati tra formati Python e JSON.
urllib Permette di scaricare dati da internet.
csv Legge e scrive file CSV.
sys Interagisce con l'interprete Python.
Esempio:
import math
print(math.sqrt(25))
# Output: 5.0

import random
print(random.randint(1, 10))
# Output: un numero casuale tra 1 e 10

Librerie esterne

Una libreria esterna è un insieme di moduli Python che non fanno parte della distribuzione standard.
Sono create da sviluppatori indipendenti o da organizzazioni per risolvere problemi specifici o fornire funzionalità avanzate.
Dove trovare le librerie esterne?
Il luogo principale per trovare e installare librerie esterne è il PyPI (Python Package Index).
Si tratta di un vasto repository online che ospita migliaia di pacchetti Python.
Puoi accedere a PyPI tramite il comando pip, lo strumento di gestione dei pacchetti incluso in Python.
Le librerie esterne più popolari includono:
Libreria Descrizione
Numpy Fondamentale per la manipolazione di array e matrici numeriche.
Pandas Offre strutture dati efficienti e strumenti per l'analisi dei dati.
Matplotlib Per la creazione di visualizzazioni grafiche.
TensorFlow e PyTorch Framework per il deep learning, utilizzati per costruire reti neurali complesse.
Django e Flask Framework web per lo sviluppo rapido di applicazioni web.
Requests Semplifica la creazione di richieste HTTP.
BeautifulSoup Per il parsing di HTML e XML.
Come installare una libreria esterna?
Utilizzare `pip` è molto semplice:
pip install nome_della_libreria
Esempio:
pip install numpy
