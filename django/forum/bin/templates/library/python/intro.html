Python

Storia e Origini

Python è stato creato da Guido van Rossum e il suo sviluppo è iniziato alla fine degli anni '80.
La prima versione ufficiale, Python 1.0, è stata rilasciata nel 1991.
Il nome "Python" deriva dalla troupe comica britannica "Monty Python", di cui van Rossum era un grande fan.

Caratteristiche principali

Leggibilità del codice: Python enfatizza una sintassi chiara e leggibile.
È spesso utilizzato come linguaggio di insegnamento grazie alla sua semplicità.
Tipizzazione Dinamica: Le variabili non richiedono una dichiarazione esplicita del tipo,
poiché il tipo viene determinato a runtime.
Interprete: è un linguaggio interpretato, il che significa che il codice viene eseguito linea per linea, facilitando il debug.
Multiparadigma: Supporta programmazione procedurale, orientata agli oggetti e funzionale.

Utilizzi e applicazioni

Sviluppo Web:Con framework come Django e Flask.
Data Science e Machine Learning:Utilizzando librerie come Pandas, NumPy, Scikit-Learn, TensorFlow e PyTorch.
Automazione e Script:Per automazione di compiti ripetitivi e scripting.
Applicazioni Desktop:Con librerie come Tkinter, PyQt e Kivy.
Elaborazione del Testo e Web Scraping:Utilizzando librerie come BeautifulSoup e Scrapy.
Giochi:Viene usato anche per la creazione di videogiochi tramite l'uso di librerie come Pygame.

Comunità

Python ha una delle comunità di sviluppo più grandi e attive. Ciò si traduce in:
Documentazione Abbondante:Guida ufficiale, tutorial, e numerosi libri.
Librerie e Framework:Un vasto repository di librerie open-source disponibili tramite il Python Package Index (PyPI)

Svantaggi

Prestazioni:Essendo interpretato, Python può essere più lento rispetto ai linguaggi compilati come C o Java
Consumo di Memoria:Può essere più pesante in termini di utilizzo della memoria.

Installazione

Molti PC e sistemi operativi hanno già installato Python.
Per verificarlo, aprire il terminale e scrivere il comando

$ python3 --version
Se il comando restituisce la versione di Python installata, allora è già presente sul sistema.
Se non è installato, è possibile scaricarlo.

Su OS Linux è possibile installare l'interprete 'python' scrivendo da CLI il seguente comando:
$ sudo apt install python3
Su OS Windows è possibile installare l'interprete tramite il seguente link: https://www.python.org/downloads/
