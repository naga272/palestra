Cpp

Storia e Origini

Il C++ è un linguaggio di programmazione general-purpose sviluppato da Bjarne Stroustrup nei primi anni '80 presso i laboratori Bell di AT&T.
La sua prima versione è stata rilasciata nel 1983 come un'estensione del linguaggio C, da cui deriva il suo nome ("C with Classes").
Stroustrup ha progettato il C++ per combinare le caratteristiche del C con nuove funzionalità di programmazione orientata agli oggetti,
facilitando la gestione di grandi progetti software.

Caratteristiche principali

Programmazione Orientata agli Oggetti (OOP): Il C++ supporta pienamente l'OOP, che consente agli sviluppatori di creare oggetti che rappresentano entità del mondo reale
Tipizzazione Statica: Le variabili devono essere dichiarate con un tipo specifico prima di essere utilizzate.
Compilato: Il codice sorgente scritto in Cpp viene tradotto in codice macchina tramite un compilatore, migliorando
le prestazioni rispetto ai linguaggi interpretati.
Gestione della Memoria: Il C++ fornisce strumenti per la gestione manuale della memoria attraverso l'uso di operatori come new e delete. Questo offre grande controllo e efficienza, ma richiede attenzione per evitare memory leak e altri problemi.
Compatibilità con il C: Il C++ è compatibile con il C, permettendo di utilizzare codice C all'interno di programmi C++ senza modifiche sostanziali.

Utilizzi e applicazioni

Sistemi Operativi: come Windows e parti di Unix/Linux, sono scritti in C++ a causa della necessità di efficienza e controllo a basso livello.
Software di Sistema: Questo include compilatori, interpreti e altri strumenti di sviluppo che richiedono un controllo preciso delle risorse di sistema.
Applicazioni Desktop: Numerosi software desktop, come Adobe Photoshop e Microsoft Office, sono scritti in C++ per sfruttare le sue prestazioni elevate.
Videogiochi: Le industrie dei videogiochi utilizzano ampiamente il C++ per sviluppare giochi complessi e ad alte prestazioni. Molti motori di gioco, come Unreal Engine, sono scritti in questo linguaggio.
Applicazioni Scientifiche e di Ingegneria: Le simulazioni scientifiche, i calcoli numerici intensivi e il software di ingegneria spesso utilizzano il C++ per la sua capacità di eseguire calcoli complessi rapidamente.
Sviluppo Embedded: Il C++ è ampiamente utilizzato nello sviluppo di software per dispositivi embedded, dove le risorse di sistema sono limitate e l'efficienza è cruciale.
Software di Rete: Molti protocolli di rete e server di alto livello sono implementati in C++ per garantire efficienza e robustezza.

Vantaggi

Prestazioni: Essendo compilato, C offre prestazioni eccellenti.
Flessibilità: Il C++ permette sia la programmazione di alto livello (astrazione) che quella di basso livello (manipolazione diretta della memoria).
Questo lo rende versatile e adatto a una vasta gamma di applicazioni.
Portabilità: Il codice C è altamente portabile tra diverse piattaforme hardware.

Svantaggi

Complessità: La gestione manuale della memoria e l'assenza di funzionalità di alto livello possono rendere Cpp complesso e incline agli errori.
Tempo di Compilazione: Il processo di compilazione del C++ può essere lento, specialmente per grandi progetti. Inoltre, l'inclusione di molti header può aumentare significativamente i tempi di compilazione.
Compatibilità tra Versioni: Le nuove versioni del C++ introducono spesso nuove funzionalità e miglioramenti, ma possono anche creare problemi di compatibilità con il codice scritto in versioni precedenti.


Installazione


Su OS Linux è possibile installare il compilatore gcc scrivendo da linea di comando il seguente comando:
$ sudo apt install gcc
Su OS Windows è possibile installare il compilatore tramite sito ufficiale sezione Download (situata a destra) > binari
oppure scaricare il programma dal seguente link: https://sourceforge.net/projects/mingw/
