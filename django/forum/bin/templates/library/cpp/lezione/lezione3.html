Variabili

Le variabili sono contenitori di dati.
In C++, dobbiamo dichiarare il tipo di variabile prima di utilizzarla.

Tipi

In C++, i tipi di variabili piu' comuni sono:
  • int: numeri interi
  • float: numeri decimali
  • double: numeri decimali con maggiore precisione
  • char: caratteri
  • string: stringhe di caratteri
  • bool: valori booleani (vero o falso)

Dichiarazione

Per dichiarare una variabile, scriviamo il tipo seguito dal nome della variabile:
int x;
Questo codice dichiara una variabile di tipo int chiamata x.
Ad essa possono essere assegnati solo numeri interi, no stringhe o caratteri o numeri decimali.
NOTA: E' raccomandato assegnare un nome significativo alle variabili per renderle piu' leggibili.
Ci sono alcune regole da seguire nella scelta dei nomi delle variabili:
  • Il nome deve iniziare con una lettera o un underscore (_)
  • Il nome puo' contenere lettere, numeri e underscores
  • Il nome e' case-sensitive (cambia tra 'Nome' e 'nome')
  • Non puo' contenere spazi o caratteri speciali
  • Non si possono usare caratteri riservati (es. int, float, cout)

Altri esempi

Ecco alcuni esempi di dichiarazione di variabili:
char b = 'a';
cout << b;
float y = 3.14;
cout << y;
string s = "Hello";
cout << s;
bool t = true;
cout << t;

Altri tipi

In C++ esistono altri tipi di variabili:
  • short: numeri interi piu' piccoli di int
  • long: numeri interi piu' grandi di int
  • long long: numeri interi piu' grandi di long
  • unsigned: numeri interi non negativi
Funzionano allo stesso modo degli altri tipi di variabili, ma con limiti diversi riguardo ai valori che possono contenere.

Costanti

Le costanti sono variabili il cui valore non puo' essere modificato.
Per dichiarare una costante, scriviamo const prima del tipo di variabile:
const int z = 10; cout << z; z = 5; cout << z;
Questo codice stampera' 10 a schermo, ma dara' errore alla riga z = 5; perche' stiamo cercando di modificare una costante.

Dichiarazione di piu' variabili Possiamo dichiarare piu' variabili dello stesso tipo in una sola riga:
int a, b, c = 5; cout << a << b << c;
