Output

Per stampare un messaggio a schermo, utilizziamo la funzione cout della libreria iostream.
Ecco un esempio:

#include using namespace std; int main(){ cout << "Hello World!"; return 0; }

Questo codice stampera' "Hello World!" a schermo.
La funzione cout e' seguita da <<, che indica l'inizio del messaggio da stampare.
Il messaggio da stampare e' racchiuso tra virgolette.
Infine, il punto e virgola ; termina l'istruzione.
Ad essa si possono concatenare pi√π messaggi separandoli con <<.
, oltre a variabili e costanti.
Ecco un esempio:

#include using namespace std; int main(){ int x = 5; cout << "Il valore di x e': " << x; return 0; }
Per andare a capo, possiamo utilizzare endl.
Ecco un esempio:

#include using namespace std; int main(){ cout << "Hello World!" << endl; return 0; }
Oppure possiamo utilizzare il carattere di escape \n.
Ecco un esempio:

#include using namespace std; int main(){ cout << "Hello World!\n"; return 0; }
