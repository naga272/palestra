Input

Per acquisire un valore da tastiera, utilizziamo la funzione cin della libreria iostream.
Ecco un esempio:

#include using namespace std; int main(){ int x; cin >> x; cout << "Hai inserito " << x; return 0; }

Questo codice acquisira' un valore da tastiera e lo stampera' a schermo.
La funzione cin e' seguita da >>, che indica l'inizio dell'acquisizione.
Il valore acquisito verra' memorizzato nella variabile x.
Infine, il punto e virgola ; termina l'istruzione.
Ad essa si possono concatenare pi√π valori separandoli con >>.
Ecco un esempio:

#include using namespace std; int main(){ int x, y; cin >> x >> y; cout << "Hai inserito " << x << " e " << y; return 0; }

Questo codice acquisira' due valori da tastiera e li stampera' a schermo.
Come esercizio, scrivete un programma che acquisisca due numeri interi da tastiera e ne stampi la somma.
NOTA: cin e cout si pronunciano "c-in" ('see-in') e "c-out" ('see-out') rispettivamente.
Essi fanno parte della libreria iostream, che sta per "input-output stream" ('flusso di input-output').
