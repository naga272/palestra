Introduzione

Creiamo il nostro primo programma in C++.
Apriamo un editor di testo e scriviamo il seguente codice:
#include using namespace std; int main(){ cout << "Hello World!"; return 0; }
Salviamo il file con estensione .cpp (es. hello.cpp) e apriamo il terminale.
C++ e' un linguaggio compilato, quindi dobbiamo compilare il nostro codice.
Per farlo, scriviamo il seguente comando:
g++ hello.cpp -o hello
Questo comando compila il nostro codice e crea un file eseguibile chiamato hello.exe
Per eseguire il programma, scriviamo il seguente comando:
./hello
Il programma stampera' "Hello World!" a schermo.
Complimenti! Avete appena scritto il vostro primo programma in C++!

Sintassi

Come molti linguaggi di programmazione, C++ ha bisogno del punto e virgola (;) alla fine di ogni istruzione.
Inoltre, il codice C++ e' case-sensitive, quindi Cout e cout sono due cose diverse.
Infine, il codice C++ deve essere racchiuso tra parentesi graffe {}.
Analizziamo il codice del nostro programma:
#include Questa riga include la libreria iostream, che ci permette di utilizzare le funzioni di input/output.
using namespace std; Questa riga ci permette di utilizzare le funzioni della libreria iostream senza doverle precedere con std::.
int main(){ Questa riga definisce la funzione principale del programma.
cout << "Hello World!"; Questa riga stampa la stringa "Hello World!" a schermo.
return 0; Questa riga termina la funzione main e restituisce il valore 0 al sistema operativo.

Commenti

I commenti sono utili per rendere il codice piu' leggibile.
In C++, i commenti possono essere di due tipi:
  • I commenti a una riga iniziano con // e terminano alla fine della riga.
  • I commenti multi-riga iniziano con /* e terminano con */.
