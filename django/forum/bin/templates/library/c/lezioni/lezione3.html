Costanti

Una costante non è altro che una variabile che una volta dichiarata non può essere più modificata.
ovviamente, come per le variabili, bisogna specificare il tipo della costante che creiamo.
Ecco un esempio:

#include <stdio.h>

const int a = 18;

int main(){
const char b = 's';
printf("costante a: %i costante b: %c", a, c);
return 0;
}

Compilazione del programma (ambiente linux)

Da command line andare nella directory corrente e scrivere il seguente comando:
$ gcc ./costanti.c -o ./costanti

Output del programma

$ ./costanti
valore costante di a: 18, valore costante di b: s

Curiosità

Cosa accade se provo a cambiare il valore di una costante?
Banalmente il compilatore darà errore, dicendo che stiamo provando a modificare una variabile che è di sola lettura.
Ecco un esempio:

const int a = 18;

int main(){
printf("costante a: %i", a);
a = 22;
printf("costante a: %i", a);
return 0;
}

Il messaggio di errore che darà durante la compilazione sarà:
$ gcc costanti.c -o ./costanti
costanti.c: In function 'main':
costanti.c:5: error: assigment of read-only variable 'a'
5 | a = 22;
