Vettori di caratteri

I vettori di caratteri, non sono altro che una sequenza di caratteri inseriti in sequenza.
La caratteristica fondamentale di un vettore è che hanno una dimensione statica, cioè che non può cambiare durante il programma.
Esiste la possibilità comunque di inizializzare un vettore senza specificarne le dimensioni, il compilatore lo capirà in automatico quando conterà il numero di elementi presenti:

#include <stdio.h>

int main(){
char vettore[] = "ciao, come stai";
printf("%s", vettore);
return 0;
}
Anche se non glie lo abbiamo detto, il vettore che abbiamo dichiarato è di grandezza 15 (caratteri) + 1(segna la fine stringa).
Infatti, tutti i vettori di caratteri che usiamo hanno sempre un carattere terminatore.
In C viene rappresentato con '\0', in python con ",0" e così via...

Output del programma

ciao, come stai

Quindi bisogna prestare attenzione alla grandezza che diamo ai vettori, perchè dobbiamo tenere conto anche del carattere di fine vettore
Se si prova a sovrascrivere, durante l'esecuzione il programma potrebbe dare errori o generare imprevisti, come i segmentation fault.
Possiamo anche semplicemente dichiarare un vettore con più spazio rispetto a quanti caratteri stiamo effettivamente usando:
#include <stdio.h>

int main(){
char vettore[200] = "ciao, come stai";
printf("%s", vettore);
return 0;
}
Il nostro nuovo array può tenere al suo interno fino a 199 caratteri, anche se effettivamente ne stiamo usando molti di meno.

Output del programma

ciao, come stai

Accedere agli elementi

Per accedere a un singolo carattere del vettore possiamo usare gli indici.
#include <stdio.h>

int main(){
char vettore[] = "ciao, come stai";
printf("singolo carattere: %c\n", vettore[1]);
return 0;
}
In questo modo stiamo dicendo che vogliamo accedere al secondo elemento del nostro vettore di caratteri.
Da notare che nella printf viene usato %c al posto di %s, questo perchè %s si usa per stampare tutto il vettore di caratteri,
mentre %c si usa solo per il singolo di carattere.


Operare sui suoi elementi

Per assegnare nuovi valori agli elementi possiamo farlo assegnando direttamente all'elemento il nuovo carattere:
#include <stdio.h>

int main(){
char vettore[] = "ciao, come stai";
vettore[0] = 's';
printf("%s\n", vettore);
return 0;
}


Output del programma

siao, come stai

E' possibile quindi lavorare su ogni singolo carattere del vettore tramite i cicli:
#include <stdio.h>

int main(){
char vettore[] = "ciao, come stai";
printf("vettore modificato: %s\n", vettore);

for(size_t i = 0; vettore[i] != '\0'; i++)
vettore[i] = 's';

printf("vettore modificato: %s\n", vettore);
return 0;
}

Output del programma

vettore modificato: ciao, come stai
vettore modificato: sssssssssssssss
