Standard Input & Output

Per standard input nella programmazione, ci si riferisce alla lettura di dati da fonti esterne, come per esempio la tastiera, file e molto altro.
In C, le librerie standard predispongono di mezzi (già costruiti da altre persone) per poter leggere del testo in diversi modi:

scanf

E' una funzione che si trova all'interno della libreria stdio.h e che consente di prendere una sola parola in ingresso (appena incontra lo spazio tronca la frase), interi, float, double ecc...
#include <stdio.h>

int main(){
char vettore[20]; // accetta un massimo di 20 caratteri il vettore
int a;
scanf("%i", &a); // per tutti i tipi la scanf vuole l'indirizzo di memoria, non il contenuto
scanf("%s", vettore); // si devono passare così, gli passiamo l'indirizzo della prima cella del vettore
return 0;
}

gets (obsoleto)

Funzione che ti consente di prendere del testo in ingresso e inserirlo all'interno di un vettore.
#include <stdio.h>

int main(){
char vettore[20];
gets(vettore); // accetta n caratteri e li inserisce nel vettore
return 0;
}
Ci tengo a precisare che questa funzione è molto rischiosa, perchè potrebbero avvenire dei problemi durante l'esecuzione del codice come segmentation-fault.
Questo perchè non si ha controllo di quanto testo viene inserito all'interno del vettore (al contrario di fgets).

fgets

Funzione che ti consente di leggere del testo in ingresso sia da file che da tastiera e memorizzarla all'interno di un vettore di caratteri.
#include <stdio.h>

int main(){
char vettore[20];
fgets(vettore, 20, stdin); // memorizza nel vettore i primi 19 caratteri scritti dall'utente
return 0;
}
Il primo parametro che passiamo alla funzione fgets è il nome del vettore, il secondo invece indica quanti caratteri vogliamo prendere in input
e il terzo indica che li voglio prendere da tastiera (stdin - stadard input), dobbiamo indicarglielo perchè ci consente appunto anche di prendere da file (lo vedremo anche più avanti)

printf

Per standard output si intende la visualizzazione a chermo di un dato
Come abbiamo già visto precedentemente, questa funzione ci consente di stampare a schermo una stringa formattata.
#include <stdio.h>

int main(){
printf("Questa è una stringa\n");
printf("stampa numeri interi %i\n", 15);
printf("stampa caratteri: %c\n", 'c');
printf("stampa numeri float: %.2\n", 2.786);
printf("stampa interi lunghi: %ld\n", 4);
return 0;
}

puts

Stampa una stringa seguita da un newline.
#include <stdio.h>

int main(){
puts("Questa è una stringa\n");
return 0;
}

putchar

Stampa un singolo carattere.
#include <stdio.h>

int main(){
putchar('c');
return 0;
}
