Standard Library

Le librerie standard si possono inlcudere all'interno del codice grazie alla direttiva "#include".
Queste librerie forniscono un set di funzioni e macro per supportare nella programmazione i programmatori.

Libreria Descrizione
stdio.h LIbreria pusata per l'input/output
stdlib.h Utile per la gestione della memoria
stdint.h Usato per la gestione di interi
string.h Usato per la gestione stringhe
time.h Usato per la gestione del tempo
locale.h Usato per l'encoding
unistd.h Libreria Unix-like usata principalemente per gestire processi
errno.h Libreria usata per la gestione errori
stdbool.h Libreria usata per la gestione booleani
stddef.h definisce alcuni tipi di variabili e macro
stdarg.h consente alle funzioni il passaggio di n argomenti come parametro
search.h permette l'uso delle tabelle hash
sys/types.h consense la definizione di tipi di variabile
ctype.h set di comandi per la classificazione dei caratteri
signal.h Usata per la gestione dei segnali a basso livello
