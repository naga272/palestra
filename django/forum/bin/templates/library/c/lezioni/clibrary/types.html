sys/types



La libreria sys/types fornisce un set di definizioni di tipi di dato utilizzabili per vari tipi di identificatori e molte grandezze che vengono usate spesso nelle chiamate di sistema di basso livello


Tipi di dato

Nome Descrizione
blkcnt_t Utilizzato per i conteggi dei blocchi di file.
blksize_t Utilizzato per le dimensioni dei blocchi.
clock_t Utilizzato per i tempi di sistema in tick di clock o CLOCKS_PER_SEC.
clockid_t Utilizzato per il tipo di ID di clock nelle funzioni di clock e timer.
dev_t Utilizzato per gli ID dei dispositivi.
fsblkcnt_t Utilizzato per i conteggi dei blocchi del file system.
fsfilcnt_t Utilizzato per i conteggi dei file nel file system.
