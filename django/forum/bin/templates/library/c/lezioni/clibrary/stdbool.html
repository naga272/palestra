stdbool



La libreria stdbool.h è un header standard del linguaggio C che definisce quattro macro per i tipi booleani.


Macro / Costanti

Nome Descrizione
true rappresenta il valore 1
false rappresenta il valore 0
__bool_true_false_are_defined serve per verificare che le definizioni di false, true e bool siano state definite.
Nel caso in cui non lo sono questa macro assume valore 0, altrimenti 1

Variabili

Nome Descrizione
bool Il compilatore sostituisce la parola bool con il tipo _Bool, che è un tipo intero che può assumere solo valore 0 oppure 1
