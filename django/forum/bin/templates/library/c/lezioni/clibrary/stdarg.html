stdarg



Per usare questa libreria, bisogna prima conoscere il concetto di "Variadic functions".
In pratica, le funzioni variadiche accettano un numero variabile di argomenti, grazie al token "...":

int somma(int, ...);
In questo modo stiamo dicendo che la funzione somma accetta obbligatoriamento un parametro intero seguito da altri n parametri
Per queste funzione è obbligatorio avere passato almeno un parametro nominato (nel nostro caso di tipo int).
Il corpo della funzione quindi sarà:
int somma(int a, ...){
// contenuto della funzione
}


Macro / Costanti

Nome Descrizione
va_start usata per iniziare l'iterazione sugli argomenti variadici di una funzione.
va_arg utilizzata per recuperare un argomento dalla lista degli argomenti variadici.
va_end utilizzata per terminare l'uso di un oggetto va_list.
va_copy copia il contenuto di una variabile va_list in un'altra.

Tipi di dato

Nome Descrizione
va_list tipo usato per iterare sugli argomenti

programma esempio che somma tutti gli argomenti passati alla chiamata fino a che non incontra il numero 0

#include <stdio.h>
#include <stdarg.h>

int somma(int, ...);

int main() {
printf("%i\n", somma(1, 2, 3, 4, 0));
printf("%i\n", somma(124, 33, 14, 0));
return 0;
}

int somma(int arg1, ...){
va_list argv;
int index,
counter = 0;

va_start(argv, arg1);
for(index = arg1; index != 0; index = va_arg(argv, int))
counter += index;
va_end(argv);
return counter;
}

Output del programma:
10
171
