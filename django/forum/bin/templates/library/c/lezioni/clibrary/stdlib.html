stdlib



La libreria stdlib sta per standard library e ha al suo interno un set di macro, procedure e funzioni di utilità generale.


Funzioni

Nome Descrizione prototype
rand Restituisce un numero intero pseudocasuale compreso tra 0 e RAND_MAX. int rand(void);
malloc,
calloc,
realloc 		funzioni che si occupano dell'allocazione dinamica della memoria void *malloc(size_t size);
void *calloc(size_t num,size_t size);
void *realloc(void* ptr, size_t size);
free Libera la memoria allocata dinamicamente dalla famiglia di funzioni malloc void free(void *ptr);
exit Causa la normale terminazione del programma. Tutte le funzioni registrate con atexit() vengono eseguite con ordine inverso rispetto alla loro registrazione, gli stream associati al programma vengono liberati, i file vengono scritti su disco (vedere flush()) ed il controllo viene restituito all'ambiente chiamante, assieme ad un valore numerico, che generalmente indica lo stato del programma o la causa della sua terminazione, che deve essere fornito alla funzione stessa. void exit(int status);
getenv Restituisce la stringa che nell'ambiente di lavoro del programma è associata al nome fornito, oppure NULL se non esiste alcuna stringa. I dettagli della funzione sono strettamente dipendenti dal sistema operativo. Bisogna guardare le variabili d'ambiente del os. char *getenv(const char *name);
system Passa la stringa fornitale all'ambiente di lavoro per l'esecuzione e restituisce il codice d'uscita del comando invocato. Se si fornisce NULL, informa sulla eventuale presenza nel sistema di un processore di comandi. int system(const char *command);
qsort Implementa in maniera generica l'algoritmo di ordinamento quicksort. void qsort(void *base, size_t nitems, size_t size, int (*compar)(const void *, const void*))
bsearch Implementa in maniera generica l'algoritmo di ricerca dicotomica. void *bsearch(const void *key, const void *base, size_t num, size_t size, int (*compar)(const void *, const void *));
abort Causa la terminazione immediata ed anormale del programma void abort(void);
atexit Registra una funzione, della quale le viene passato il puntatore, affinché venga eseguita appena prima della normale terminazione del programma. int atexit(void (*func)(void));
atof Converte una stringa in un numero in virgola mobile. Equivalente a strtod(s, (char**)NULL). double atof(const char *str);
atoi Converte una stringa in un numero intero. int atoi(const char *str);
atol Converte una stringa in un numero intero lungo (long int) long atol(const char *str);
strtod Converte una stringa in un double (numero a virgola mobile), effettuando dei controlli sull'overflow e restituendo anche l'eventuale parte non convertita della stringa. double strtod(const char *str, char **endptr);
strtol La funzione strtol (string to long) converte una stringa che rappresenta un numero in formato intero lungo (long) e fornisce un puntatore alla parte non convertita della stringa. long strtol(const char *str, char **endptr, int base);
strtoul La funzione strtoul (string to unsigned long) converte una stringa che rappresenta un numero in formato intero senza segno (unsigned long) e fornisce un puntatore alla parte non convertita della stringa. unsigned long strtoul(const char *str, char **endptr, int base);
div
ldiv 		div viene usata per dividere un numeratore con il denominatore, mentre ldiv fa la stessa cosa ma con interi lunghi div_t div(int numer, int denom);
ldiv_t ldiv(long numer, long denom);
abs
labs 		Calcola il valore assoluto dell'argomento. int abs(int n);
long labs(long n);

Macro / Costanti

Nome Descrizione
NULL Una macro che espande nella costante puntatore nullo; in altre parole, una costante che rappresenta un valore che è garantito essere l'indirizzo di una posizione non valida nella memoria.
EXIT_FAILURE Indica una terminazione non corretta del programma: viene generalmente utilizzata assieme alla funzione exit() (ha valore 1).
EXIT_SUCCESS Indica una terminazione corretta del programma: viene generalmente utilizzata assieme alla funzione exit() (ha valore 0).
RAND_MAX Massimo valore restituibile dalla funzione rand() (>= 32767)

Tipi di dato

Nome Descrizione
size_t Un tipo intero che è il tipo del valore restituito dall'operatore sizeof.
div_t e ldiv_t Una struttura contenente le informazioni restituite dalle funzioni div e ldiv.
