Variabili

Una variabile non è altro che uno spazio di memoria che contiene un valore. Il C è un linguaggio fortemente tipato, questo significa che il programmatore è tenuto a specificare il tipo di ogni variabile che creiamo.

Esempio di Dichiarazione di Variabili

int a = 18;
int main() {
int b = 5;
return 0;
}

Analisi del Programma

Nei programmi esistono due tipologie di variabili: locali e globali.

  • Variabili Globali: Possono essere viste da tutto il programma e sono esterne alle funzioni.
  • Variabili Locali: Sono visibili solo all'interno delle funzioni.

Nel programma sopra, la variabile a è globale, mentre la variabile b è locale (quindi non visibile fuori dal main). Le variabili dichiarate sono di tipo int, che significa che contengono solo numeri interi.

Esempi di Codice

Uso della Libreria stdint.h

#include <stdint.h>

uint8_t a = 18;

int main() {
int32_t b = 5;
return 0;
}

Variabili di Tipo Char

#include <stdio.h>

int main() {
char a = 'b';
printf("%c", a);
return 0;
}

In questo programma viene dichiarata una variabile di tipo char (carattere) e gli viene assegnato il carattere 'b'. È importante notare che sono stati usati i singoli apici per assegnare il carattere e non i doppi apici.

Stampa di Più Tipi di Variabili

#include <stdio.h>

int main() {
printf("carattere: %c; intero: %i", 'b', 58);
return 0;
}

L'output che otteniamo dal programma è il seguente:

carattere: b; intero: 58

Cast di Tipo

Durante l'esecuzione del programma è possibile cambiare il tipo della variabile. Questo metodo viene detto "cast di tipo" e si sviluppa con la seguente sintassi:

(new_type) nome_variabile;

Esempio pratico di cast di tipo:

#include <stdio.h>

int main() {
int a = 97;
printf("%c\n", (char) a);
return 0;
}

L'output che otteniamo dal programma è il seguente:

a

Perché otteniamo questo risultato? Per la macchina, i caratteri non sono altro che numeri interi, quindi quando noi scriviamo un carattere la macchina lo interpreta come un numero intero univoco che identifica un certo carattere.
