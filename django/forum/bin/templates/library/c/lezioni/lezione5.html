Costrutti selettivi

I costrutti selettivi servono per eseguire del codice solo se all'interno del programma vengono rispettate determinate condizioni.
In C, basta usare la parola speciale "if" ed "else" nel seguente modo:
int main(){
if(condizione){
// esegui questo codice
}else{
// fai altro
}
return 0;
}
Se la condizione risulta vera (cioè il risultato è diverso da 0), allora verrà eseguito il codice all'interno dell'if, altrimenti eseguirà il codice all'interno dall'else.
Programma pratico:
#include <stdio.h>

int main(){
int a;
if(4 > 2){
a = 4;
}else{
a = 8;
}
printf("%d", a);
return 0;
}

Output del programma

$ valore di a: 4

Analisi del programma sorgente.c

Nel programma dichiariamo una variabile di tipo intero chiamata a.
Dopodichè usiamo il costrutto condizionale, che determinerà il valore di a in base al risultato della condizione.
Se la condizione risulta vera allora la variabile avrà come valore 4, altrimenti avrà come valore 8.
E' possibile estendere le possibilità di condizioni, usando parola chiave "else if":
#include <stdio.h>

int main(){
int a;
if(4 == 2){
a = 4;
}else if(10 > 3){
a = 8;
}else{
a = 0;
}
printf("valore di a: %d", a);
return 0;
}
In questo programma, se la condizione del primo if non è vera, passerà alla condizione dell'else if che se anche questa non è verà, andrà a eseguire l'else.
Nel nostro caso, l'else if è vera perchè la sua condizione restituisce il valore 1 (al contrario della condizione dell'if), quindi l'output sarà:
$ valore di a: 8

metodi alternativi per scrivere condizioni

Il linguaggio C ci consente di strutturare in modi diversi le nostre condizioni.
Come abbiamo visto prima, affianco all'if era presente la condizione e le graffe che racchiudevano tutto il codice che si sarebbe eseguito solo se la condizione venisse soddisfatta.
Esistono altri modi per scrivere questo, guardiamone alcuni:
#include <stdio.h>

int main(){
int a;
if(4 == 2)
a = 4;
else if(10 > 3)
a = 8;
else
a = 0;
printf("valore di a: %d", a);
return 0;
}
E' lo stesso codice di prima, con l'unica differenza che non ci sono le graffe.
Quando l'if, l'else e l'else if contengono una sola riga di codice, è possibili usare la sintassi sopra citata.
Un'altro metodo per scrivere una condizione è la seguente:
#include <stdio.h>

int main(){
int a = (10 > 3)? 4 : 5;
printf("valore di a: %d", a);
return 0;
}
In questo programma assegnamo un valore alla variabile "a" a seconda del risultato della condizione.
"E' 10 maggiore di 3? se sì, assegno alla variabile a il numero 4, altrimenti assegno il 5".
ricapitolando, la sintassi per usare questa condizione è:
(condizione)? espressione1 : espressione2;
E anche possibile introdurre questo tipo di if all'interno delle funzioni:
#include <stdio.h>

int main(){
printf("valore: %d", (10 > 3)? 4 : 5);
return 0;
}

Costrutto switch case

Lo switch è una struttura di controllo che permette di selezionare una delle diverse possibili esecuzioni del codice in base al valore di una variabile.
E' utile per confrontare molte condizioni e per rendere il codice molto più leggibile e elegante rispetto a una serie di else if.

#include <stdio.h>

int main(){
int a = 5;
switch(a){
case 1:
printf("la variabile a vale 1\n");
break;
case 10:
printf("la variabile a vale 10\n");
break;
case 5:
printf("la variabile a vale 5\n");
break;
default:
printf("la variabile a ha un valore diverso rispetto ai casi descritti\n");
break;
}
return 0;
}
In questo programma, vediamo l'uso dello switch, in pratica in base al valore della variabile a, eseguirà un determinato flusso di codice, che nel nostro caso, eseguirà il flusso all'interno di "case 5" perchè la nostra variabile a ha come valore 5.
L'istruzione break consente di forzare l'interruzione dell'esecuzione dello switch, in modo tale che non vada a eseguire altre condizioni.
Se il valore di a non corrisponde a nessuno dei casi specificati, verrà eseguito il blocco default, che stampa "la variabile a ha un valore diverso rispetto ai casi descritti".
Anche qui si usa break per interrompere l'esecuzione dello switch.

Output

la variabile a vale 5


Costrutto iteratore while

In un qualunque linguaggio di programmazione Turing completo, è presente almeno un ciclo while.
I cicli servono per ripetere un set di istruzioni fintanto che una condizione non viene soddisfatta.
Esempio:

#include <stdio.h>

int main(){
int a = 5;
while(a < 10){
printf("Hello World\n");
}
return 0;
}


In questo esempio, viene mostrato un programma dove viene usato un ciclo while.
Fin tanto che la variabile "a" è minore di 10, stampa a schermo la scritta "Hello World", altrimenti vai avanti col programma.
Come avrete notato, questo programma non terminerà mai, questo perchè la variabile "a" non cambia mai il suo valore, rimane sempre minore di 10.
Quindi come possiamo fare?
Ci basta semplicemente cambiare il valore di "a" all'interno del ciclo:
#include <stdio.h>

int main(){
int a = 5;
while(a < 10){
printf("Hello World\n");
a++;
}
return 0;
}
Seguiamo la logica del programma passo passo:
dichiariamo una variabile intera chiamata a e gli assegnamo il valore 5.
dopodichè, abbiamo il while con la sua condizione, "è il valore di a minore di 10?" sì perchè ha valore 5, quindi stampiamo a schermo la scritta "Hello World" e incrementiamo di 1 la variabile a.
Ora il programma ritorna alla condizione, è il valore di a minore di 10? sì, perchè adesso la variabile a ha come valore 6 e quindi ristampa a schermo "Hello World".
Quindi riesegue le istruzioni all'interno del ciclo, finchè la variabile a non diventa uguale o maggiore di 10
Output del programma:
Hello World
Hello World
Hello World
Hello World
Hello World

Quindi ricapitolando la sintassi:
while(condizione){
// istruzioni interne al ciclo
}


Costrutto iteratore for

Un ciclo for in C è usato per ripetere un blocco di codice un numero specifico di volte.
La sua struttura è composta da tre parti principali: inizializzazione, condizione, e aggiornamento.
Sintassi base:
for(inizializzazione; condizione; aggiornamento){
// istruzioni interne al ciclo
}


  • inizializzazione: Questa parte viene eseguita solo una volta all'inizio del ciclo.
  • Condizione: Prima di ogni iterazione del ciclo, viene verificata questa condizione.
    Se la condizione è vera, il ciclo continua; se è falsa, il ciclo termina.
  • aggiornamento: Dopo ogni iterazione del ciclo, viene eseguito l'aggiornamento

Ecco un esempio per capire meglio come usare il ciclo for:
#include <stdio.h>

int main(){
for(int i = 0; i < 5; i++){
printf("valore di i: %d\n", i);
}
return 0;
}

Output del programma

Valore di i: 0
Valore di i: 1
Valore di i: 2
Valore di i: 3
Valore di i: 4

Costrutto do-while

Il costrutto do-while consente di ripetere un blocco di codice fino a quando una condizione specificata è vera.
La caratteristica principale del ciclo do-while è che il blocco di codice viene eseguito almeno una volta, indipendentemente dal fatto che la condizione sia vera o falsa all'inizio.
Sintassi:
do{
// istruzioni
}while(condizione);
